
# Secure Survey Postback System (FastAPI)

This repository demonstrates how to implement a secure 
server-to-server postback system for survey reward platforms.

## Features
- SHA-256 signature verification
- Duplicate transaction prevention
- Atomic reward crediting
- Fraud-safe structure

## Live Example
This implementation is tested in a real-world rewards platform:
https://earnvra.in

## Tech Stack
- FastAPI
- PostgreSQL
- Async request handling
